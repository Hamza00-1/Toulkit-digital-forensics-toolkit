1. The Challenge in Modern Forensics

  • Scale & Noise: Investigators must manually parse thousands of log lines and hidden metadata tags.
  • Data Mutilation: Suspects format drives, delete files, and obfuscate payloads to hide tracks.
  • Collaboration Bottlenecks: Heavy offline processing environments make remote reporting and stakeholder updates difficult.

Goal: A solution that balances heavy offline processing with agile, secure reporting.

2. Aegis Architecture: Desktop vs. Web

Python Desktop Client (Offline)

  • Built with CustomTkinter for a responsive, modern UI.
  • Deep OS integration for raw byte carving.
  • Ensures evidence never leaks to the internet.

Flask Web Dashboard (Remote)

  • Accessible via browser for non-technical managers.
  • Triage reports and analytics.
  • Secure RESTful API mapping.

3. Integrity & Chain of Custody

Principle: If evidence is altered without a log, it is inadmissible in court.

  • Every action (metadata extraction, hashing, carving) triggers an automated INSERT into a secure MySQL Activity Tracker Database.
  • Captures timestamps, target files, success states, and the specific module used.
  • Creates an immutable, mathematically sound audit trail for the investigator.

4. Validation Over Trust: Threat Intel

  • Mathematical Proof: Utilizes the hashlib library to compute MD5, SHA-1, and SHA-256 signatures in safe chunks, guaranteeing file integrity.
  • Global Threat Intelligence: Integrates directly with the VirusTotal API.
  • Instead of relying on local assumptions, we query global heuristic databases to flag known malware (like the EICAR test string) instantly.

5. Automating the Triage

The Sentinel Syslog Abstractor minimizes human error:

  • Regex Pattern Matching: We utilize the re library to extract timestamps, hostnames, and messages from standard auth.log formats.
  • Data Structuring: We map this unstructured text into Pandas DataFrames.
  • Anomaly Detection: The system automatically flags IPs with multiple failed SSH login attempts, highlighting brute-force vectors without manual reading.

6. Navigating the Messy Reality

Digital evidence is rarely pristine. Aegis includes tools for recovering damaged data:

  • Hexadecimal Data Carving: Bypasses corrupted file systems entirely. Scans raw memory dumps and byte-streams to isolate files using explicitly defined "Magic Headers".
  • Payload Decoder Matrix: Translates suspicious Base64, Hexadecimal, and Binary strings back into human-readable text using binascii.
